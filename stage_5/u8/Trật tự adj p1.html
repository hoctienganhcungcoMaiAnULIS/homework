📚 Lý Thuyết: Công Thức Trật Tự Tính Từ

✨ Thứ tự đầy đủ (từ trái → phải):
Từ hạn định
(a, the, my...) → Số lượng
(one, two...) → Ý kiến
(nice, lovely...) → Kích thước
(big, small...) → Tình trạng
(clean, rich...) → Tuổi tác
(old, new...) → Hình dáng
(round, fat...) → Màu sắc
(red, blue...) → Nguồn gốc
(Italian, French...) → Chất liệu
(wooden, silk...) → Mục đích
(riding, sleeping...) → DANH TỪ
🧠 Câu thần chú ghi nhớ:
"Ông Sáu Cần Ăn Súp Cua Ở Mỹ Phải"
Ông = Opinion (Ý kiến) Sáu = Size (Kích thước) Cần = Condition (Tình trạng) Ăn = Age (Tuổi tác) Súp = Shape (Hình dáng) Cua = Color (Màu sắc) Ở = Origin (Nguồn gốc) Mỹ = Material (Chất liệu) Phải = Purpose (Mục đích)
Thứ tự Loại tính từ Tiếng Việt Ví dụ

4AgeTuổi tác / Thời gianold, young, new, ancient, antique, modern
5ShapeHình dánground, square, long, fat, thin
6ColorMàu sắcgreen, blue, red, white, purple, reddish
7OriginNguồn gốc / Xuất xứAmerican, British, Italian, French, Japanese
8MaterialChất liệugold, wooden, plastic, cotton, silk, leather
