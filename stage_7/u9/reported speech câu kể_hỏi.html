🌺 Lý Thuyết Tường Thuật (Reported Speech)

1. Quy tắc Lùi thì (Tense Changes)

Trực tiếp (Direct)Gián tiếp (Reported)
Hiện tại đơn (V1/Vs,es)Quá khứ đơn (V2/Ved)
Hiện tại tiếp diễn (am/is/are + Ving)Quá khứ tiếp diễn (was/were + Ving)
Quá khứ đơn (V2/Ved)Quá khứ hoàn thành (had + V3/Ved)
Hiện tại hoàn thành (have/has + V3/ed)Quá khứ hoàn thành (had + V3/Ved)
Tương lai đơn (will)Tương lai trong quá khứ (would)
Can / May / MustCould / Might / Had to

2. Quy tắc đổi Trạng từ (Time & Place Expressions)

Trực tiếpGián tiếp
Today / TonightThat day / That night
TomorrowThe next day / The following day
YesterdayThe day before / The previous day
Ago / Now / Here / ThisBefore / Then / There / That

3. Cấu trúc câu hỏi và mệnh lệnh

🌸 Câu hỏi Yes/No: S + asked (sb) + if/whether + S + V (lùi thì).
🌸 Câu hỏi Wh-: S + asked (sb) + Wh-word + S + V (lùi thì - tuyệt đối không đảo tobe/trợ động từ lên trước).
🌸 Câu mệnh lệnh/nhờ vả: S + asked/told + sb + (not) to + V.
